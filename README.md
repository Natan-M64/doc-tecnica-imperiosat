# Documentação Técnica Interna - Impériosat

Este repositório foi criado como parte da Atividade Extensionista II do curso de Análise e Desenvolvimento de Sistemas da UNINTER.

## Objetivo do projeto

O objetivo do projeto é organizar um modelo padronizado de documentação técnica interna para a empresa Impériosat, localizada em Juru - PB, facilitando o acesso às informações e apoiando processos de capacitação dos colaboradores.

## Tecnologias e ferramentas utilizadas

1. GitHub
2. Google Drive/Docs
3. HTML e CSS
4. Markdown

## Documentos Disponíveis

- Modelo de Procedimento Técnico
- Modelo de Stack Docker
- Modelo de Backup e Restauração
- Política de Documentação Segura
